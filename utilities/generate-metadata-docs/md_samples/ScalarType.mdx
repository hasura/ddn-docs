### ScalarType

Definition of a user-defined scalar type that that has opaque semantics.

**Properties**

| Name | Type | Required | Description |
| ----- | ----- | ----- | ----- |
| `name` | `string` | Yes | The name to give this scalar type, used to refer to it elsewhere in the metadata. Must be unique across all types defined in this subgraph. |
| `graphql` | [`object`](#scalartypegraphqlconfiguration) | No | Configuration for how this scalar type should appear in the GraphQL schema. |
| `description` | `string` | No | The description of this scalar. Gets added to the description of the scalar's definition in the graphql schema. |


**Example:**

```yaml
name: CustomString
graphql:
  typeName: CustomString
description: A custom string type

```


#### ScalarTypeGraphQLConfiguration

GraphQL configuration of an Open DD scalar type



**Properties**

| Name | Type | Required | Description |
| ----- | ----- | ----- | ----- |
| `typeName` | `string` | Yes | The name of the GraphQl type to use for this scalar. |
