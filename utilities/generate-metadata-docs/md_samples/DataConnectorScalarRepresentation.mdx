### DataConnectorScalarRepresentation

The representation of a data connector scalar in terms of Open DD types

**Properties**

| Name | Type | Required | Description |
| ----- | ----- | ----- | ----- |
| `dataConnectorName` | `string` | Yes | The name of the data connector that this scalar type comes from. |
| `dataConnectorScalarType` | `string` | Yes | The name of the scalar type coming from the data connector. |
| `representation` | `string` | Yes | The name of the Open DD type that this data connector scalar type should be represented as. |
| `graphql` | [`object`](#dataconnectorscalargraphqlconfiguration) | No | Configuration for how this scalar's operators should appear in the GraphQL schema. |


**Example:**

```yaml
dataConnectorName: data_connector
dataConnectorScalarType: varchar
representation: String
graphql:
  comparisonExpressionTypeName: String_Comparison_Exp

```


#### InbuiltType

An inbuilt primitive OpenDD type.

This can be one of the following:

<div><pre><code>ID | Int | Float | Boolean | String</code></pre></div>
