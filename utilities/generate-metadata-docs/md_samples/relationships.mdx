### Relationship

Definition of a relationship on an OpenDD type which allows it to be extended with related models or commands.

| Name         | Type   | Required | Description                                                                                                    |
| ------------ | ------ | -------- | -------------------------------------------------------------------------------------------------------------- |
| `kind`       | string | true     |                                                                                                                |
| `version`    | string | true     |                                                                                                                |
| `definition` | object | true     | Definition of a relationship on an OpenDD type which allows it to be extended with related models or commands. |

| Name          | Type        | Required | Description                                                                                                   |
| ------------- | ----------- | -------- | ------------------------------------------------------------------------------------------------------------- |
| `name`        | object      | true     | The name of the relationship.                                                                                 |
| `source`      | object      | true     | The source type of the relationship.                                                                          |
| `target`      | object      | true     | The target of the relationship.                                                                               |
| `mapping`     | array       | true     | The mapping configuration of source to target for the relationship.                                           |
| `description` | string,null | false    | The description of the relationship. Gets added to the description of the relationship in the graphql schema. |

Examples:

```yaml
name: Articles
source: author
target:
  model:
    name: Articles
    namespace: null
    subgraph: null
    relationshipType: Array
mapping:
  - source:
      fieldPath:
        - fieldName: author_id
    target:
      modelField:
        - fieldName: author_id
description: Articles written by an author
```

#### RelationshipTarget

The target for a relationship.

Examples:

```yaml
model:
  name: Articles
  namespace: null
  subgraph: null
  relationshipType: Array
```

| Name    | Type   | Required | Description                          |
| ------- | ------ | -------- | ------------------------------------ |
| `model` | object | true     | The target model for a relationship. |

#### ModelRelationshipTarget

The target model for a relationship.

| Name               | Type        | Required | Description                                                         |
| ------------------ | ----------- | -------- | ------------------------------------------------------------------- |
| `name`             | object      | true     | The name of the data model.                                         |
| `namespace`        | string,null | false    |                                                                     |
| `subgraph`         | string,null | false    | The subgraph of the target model. Defaults to the current subgraph. |
| `relationshipType` | object      | true     | Type of the relationship - object or array.                         |

#### RelationshipType

Type of the relationship.

| Name      | Type   | Required | Description                            |
| --------- | ------ | -------- | -------------------------------------- |
| `command` | object | true     | The target command for a relationship. |

#### CommandRelationshipTarget

The target command for a relationship.

| Name       | Type        | Required | Description                                                           |
| ---------- | ----------- | -------- | --------------------------------------------------------------------- |
| `name`     | object      | true     | The name of the command.                                              |
| `subgraph` | string,null | false    | The subgraph of the target command. Defaults to the current subgraph. |

#### RelationshipMapping

Definition of a how a particular field in the source maps to a target field or argument.

| Name     | Type   | Required | Description                                             |
| -------- | ------ | -------- | ------------------------------------------------------- |
| `source` | object | true     | The source configuration for this relationship mapping. |
| `target` | object | true     | The target configuration for this relationship mapping. |

Examples:

```yaml

  fieldPath:
    - fieldName: author_id

  modelField:
    - fieldName: author_id
```
