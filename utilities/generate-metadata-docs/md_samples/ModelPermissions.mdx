### ModelPermissions

Definition of permissions for an OpenDD model.

**Properties**

| Name          | Type                        | Required | Description                                                    |
| ------------- | --------------------------- | -------- | -------------------------------------------------------------- |
| `modelName`   | `string`                    | Yes      | The name of the model for which permissions are being defined. |
| `permissions` | [`array`](#modelpermission) | Yes      | A list of model permissions, one for each role.                |

**Example:**

```yaml
modelName: Articles
permissions:
  - role: admin
    select:
      filter: null
  - role: user
    select:
      filter:
        fieldComparison:
          field: author_id
          operator: _eq
          value:
            sessionVariable: x-hasura-user-id
```

#### ModelPermission

Defines the permissions for an OpenDD model.

**Properties**

| Name     | Type                          | Required | Description                                                                                                               |
| -------- | ----------------------------- | -------- | ------------------------------------------------------------------------------------------------------------------------- |
| `role`   | `string`                      | Yes      | The role for which permissions are being defined.                                                                         |
| `select` | [`object`](#selectpermission) | No       | The permissions for selecting from this model for this role. If this is null, the role is not allowed to query the model. |

**Example:**

```yaml
role: user
select:
  filter:
    fieldComparison:
      field: author_id
      operator: _eq
      value:
        sessionVariable: x-hasura-user-id
```

#### SelectPermission

Defines the permissions for selecting a model for a role.

**Properties**

| Name     | Type   | Required | Description                                                                                        |
| -------- | ------ | -------- | -------------------------------------------------------------------------------------------------- |
| `filter` | booboo | Yes      | Filter expression when selecting rows for this model. Null filter implies all rows are selectable. |

#### ModelPredicate

A predicate that can be used to restrict the objects returned when querying a model.

**Properties**

| Name              | Type   | Required | Description                                                                                         |
| ----------------- | ------ | -------- | --------------------------------------------------------------------------------------------------- |
| `fieldComparison` | object | Yes      | Field comparision predicate filters objects based on a field value.                                 |
| `fieldIsNull`     | object | Yes      |                                                                                                     |
| `relationship`    | object | Yes      | Relationship predicate filters objects of a source model based on a predicate on the related model. |
| `And`             | object | Yes      | Evaluates to true if all sub-predicates evaluate to true.                                           |
| `Or`              | object | Yes      | Evaluates to true if any of the sub-predicates evaluate to true.                                    |
| `Not`             | object | Yes      | A predicate that can be used to restrict the objects returned when querying a model.                |

**Example:**

```yaml
fieldComparison:
  field: author_id
  operator: _eq
  value:
    sessionVariable: x-hasura-user-id
```
