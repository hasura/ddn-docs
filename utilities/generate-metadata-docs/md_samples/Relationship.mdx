### Relationship

Definition of a relationship on an OpenDD type which allows it to be extended with related models or commands.

**Properties**

| Name | Type | Required | Description |
| ----- | ----- | ----- | ----- |
| `name` | `string` | Yes | The name of the relationship. |
| `source` | `string` | Yes | The source type of the relationship. |
| `target` | [`object`](#relationshiptarget) | Yes | The target of the relationship. |
| `mapping` | [`array`](#relationshipmapping) | Yes | The mapping configuration of source to target for the relationship. |
| `description` | `string,null` | No | The description of the relationship. Gets added to the description of the relationship in the graphql schema. |


**Example:**

```yaml
name: Articles
source: author
target:
  model:
    name: Articles
    namespace: null
    subgraph: null
    relationshipType: Array
mapping:
  - source:
      fieldPath:
        - fieldName: author_id
    target:
      modelField:
        - fieldName: author_id
description: Articles written by an author

```


#### RelationshipTarget

The target for a relationship.



**Properties**

| Name | Type | Required | Description |
| ----- | ----- | ----- | ----- |
| `model` | object | Yes | The target model for a relationship. |
| `command` | object | Yes | The target command for a relationship. |


**Example:**

```yaml
model:
  name: Articles
  namespace: null
  subgraph: null
  relationshipType: Array

```


#### RelationshipMapping

Definition of a how a particular field in the source maps to a target field or argument.



**Properties**

| Name | Type | Required | Description |
| ----- | ----- | ----- | ----- |
| `source` | [`object`](#relationshipmappingsource) | Yes | The source configuration for this relationship mapping. |
| `target` | [`object`](#relationshipmappingtarget) | Yes | The target configuration for this relationship mapping. |


**Example:**

```yaml
source:
  fieldPath:
    - fieldName: author_id
target:
  modelField:
    - fieldName: author_id

```


#### RelationshipMappingSource

The source configuration for a relationship mapping.



**Properties**

| Name | Type | Required | Description |
| ----- | ----- | ----- | ----- |
| `SourceValue` | object | Yes | An expression which evaluates to a value that can be used in permissions. |
| `SourceField` | object | Yes |  |


#### RelationshipMappingTarget

The target configuration for a relationship mapping.



**Properties**

| Name | Type | Required | Description |
| ----- | ----- | ----- | ----- |
| `TargetArgument` | object | Yes | N/A |
| `TargetModelField` | object | Yes |  |
