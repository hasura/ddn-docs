# Data Modeling

## Introduction

Hasura DDN uses a code-first, semantic metadata layer to centralize your data definitions, relationships, and
permissions in one place. This setup makes it easy to organize and scale your data model, whether you're building a
simple app or a complex architecture. With Hasura, turning data into APIs is fast and straightforward.
