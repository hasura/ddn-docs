# Open DDS reference: modelGraphqlApi

## Introduction

To enable GraphQL API on a model, `modelGraphqlApi` Open DDS object is required to be defined.
As of now, the following query APIs are supported.
- Select Many
- Select Uniques

## JSON schema

The JSON schema for `modelGraphqlApi` needs to specify GraphQL field names and type names
against respective query type.

The following is the JSON schema of the `modelGraphqlApi` type.

```json
{
  "description": "The definition of the GraphQL API component specific to a model.",
  "type": "object",
  "required": [
    "kind",
    "modelName",
    "selectUniques"
  ],
  "properties": {
    "argumentsInputField": {
      "anyOf": [
        {
          "$ref": "#/definitions/GraphQlFieldName"
        },
        {
          "type": "null"
        }
      ]
    },
    "argumentsInputType": {
      "anyOf": [
        {
          "$ref": "#/definitions/GraphQlTypeName"
        },
        {
          "type": "null"
        }
      ]
    },
    "filterExpressionType": {
      "description": "The type name of the filter boolean expression.",
      "anyOf": [
        {
          "$ref": "#/definitions/GraphQlTypeName"
        },
        {
          "type": "null"
        }
      ]
    },
    "kind": {
      "type": "string",
      "enum": [
        "modelGraphqlApi"
      ]
    },
    "modelName": {
      "$ref": "#/definitions/ModelName"
    },
    "selectMany": {
      "anyOf": [
        {
          "$ref": "#/definitions/SelectManyGraphQlDefinition"
        },
        {
          "type": "null"
        }
      ]
    },
    "selectUniques": {
      "type": "array",
      "items": {
        "$ref": "#/definitions/SelectUniqueGraphQlDefinition"
      }
    }
  }
}
```

### Related objects

#### SelectManyGraphQlDefinition

```json
{
   "description": "The definition of the GraphQL API for selecting rows from a model.",
   "type": "object",
   "required": [
     "queryRootField"
   ],
   "properties": {
     "queryRootField": {
       "description": "The name of the query root field for this API.",
       "allOf": [
         {
           "$ref": "#/definitions/GraphQlFieldName"
         }
       ]
     }
   }
}
```

#### SelectUniqueGraphQlDefinition
```json
{
  "description": "The definition of the GraphQL API for selecting a unique row/object from a model.",
  "type": "object",
  "required": [
    "queryRootField",
    "uniqueIdentifier"
  ],
  "properties": {
    "queryRootField": {
      "description": "The name of the query root field for this API.",
      "allOf": [
        {
          "$ref": "#/definitions/GraphQlFieldName"
        }
      ]
    },
    "uniqueIdentifier": {
      "description": "A set of fields which can uniquely identify a row/object in the model.",
      "type": "array",
      "items": {
        "$ref": "#/definitions/FieldName"
      }
    }
  }
}
```

#### GraphQlFieldName

```json
{
  "description": "The name of a GraphQL object field.",
  "type": "string"
}
```

#### GraphQlTypeName

```json
{
  "type": "string"
}
```

#### ModelName

```json
{
  "description": "The name of data model.",
  "type": "string"
}
```

#### FieldName

```json
{
  "description": "The name of a field in a user-defined object type.",
  "type": "string"
}
```
