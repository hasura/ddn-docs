# Select Many

## Introduction

You can fetch multiple nodes of the same type using a select many query.

query:
```graphql
query {
    ArticleMany(limit: 1){
      article_id
      title
    }
}
```
response:
```json
{
  "ArticleMany": [
    {
      "article_id": 1,
      "title": "The Next 700 Programming Languages"
    }
  ]
}
```

## Open DDS Metadata

You need to define `selectMany` property in [`modelGraphqlApi`](/api-schema/queries/open-dds-reference.mdx#open-dds-reference-modelgraphqlapi) open DDS object.
It is required to specify root field name of select many query. See [`SelectManyGraphQlDefinition`](/api-schema/queries/open-dds-reference.mdx#selectmanygraphqldefinition) reference for more details.

A sample metadata for `Articles` model:
```json
{
    "kind": "modelGraphqlApi",
    "modelName": "Articles",
    "selectUniques": [
        {
            "queryRootField": "ArticleByID",
            "uniqueIdentifier": [
                "article_id"
            ]
        }
    ],
    "selectMany": {
        "queryRootField": "ArticleMany"
    }
}
```

## Filter Expression

Using `filterExpressionType` property in `modelGraphqlApi` metadata object, you can define the type name for
filter expression input object type. However, it is optional and defaults to `{ModelName}_bool_exp` format.

GraphQL API metadata for `Articles` model with filter expression type:
```json
{
    "kind": "modelGraphqlApi",
    "modelName": "Articles",
    "selectUniques": [
        {
            "queryRootField": "ArticleByID",
            "uniqueIdentifier": [
                "article_id"
            ]
        }
    ],
    "selectMany": {
        "queryRootField": "ArticleMany"
    },
    "filterExpressionType": "Article_Where_Exp"
}
```

Example query:
```graphql
query {
    AuthorMany(where: {_or: [{first_name: {like: "Pet"}}]}) {
      author_id first_name
    }
}
```
response:
```json
{
  "AuthorMany": [
    {
      "author_id": 1,
      "first_name": "Peter"
    }
  ]
}
```

The above metadata generates the following graphql schema.

```graphql
type Article {
  article_id: Int!
  author_id: Int!
  title: String!
}

input Article_Where_Exp {
  _and: [Article_Where_Exp!]
  _not: Article_Where_Exp
  _or: [Article_Where_Exp!]
  article_id: db_Int_comparison_exp
  author_id: db_Int_comparison_exp
  title: String_Comparison_Exp
}

input db_Int_comparison_exp {
  _eq: Int
  _is_null: Boolean
}

interface Node {
  id: ID!
}

type Query {
  ArticleByID(article_id: Int!): Article
  ArticleMany(limit: Int, offset: Int, where: Article_Where_Exp): [Article!]!
  node(id: ID!): Node
}

input String_Comparison_Exp {
  _eq: String
  _is_null: Boolean
  like: String
}
```
