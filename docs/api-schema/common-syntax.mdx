### FieldComparison

```json
{
  "fieldComparison": {
    "field": "<FieldName>",
    "operator": "<Operator>",
    "value": <ValueExpression>
  }
}
```

- `FieldName`: name of the field of the model to compare (as `String`)
- `Operator`: name of the operator. Either the built-in operator `_eq` or `_is_null` or any of the operators available
  from the data source.
- `ValueExpression`: [ValueExpression](#valueexpression)
