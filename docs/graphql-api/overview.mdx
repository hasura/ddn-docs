# Hasura GraphQL API: Overview

<div className="overview-header">
  <div className="overview-text">
    <p>
      Hasura offers a GraphQL API tailored to your data domain. To set it up, you simply need to create a DDS object with the
      type <code>modelGraphqlApi</code> corresponding to a model. Hasura takes care of auto-generating the GraphQL schema based on the
      definitions of your data domain's models and types.
    </p>
    <p>
      Additionally, the <code>modelGraphqlApi</code> DDS object provides the
      flexibility to customize specific aspects of your GraphQL schema. Using <code>scalarTypeGraphqlExpression</code> DDS object, you
      can customize the comparison input objects for any data source's scalar types.
    </p>
