# Delete Data

## Update an object by a unique field

You can delete an object by targeting it using a unique key according to the source schema.

<GraphiQLIDE
  query={`mutation DeletePost {
  deletePostsById(keyId: "6") {
    affectedRows
  }
}`}
  response={`{
  "data": {
    "deletePostsById": {
      "affectedRows": 1
    }
  }
}`}
/>
