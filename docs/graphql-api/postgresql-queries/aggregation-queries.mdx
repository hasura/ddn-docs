# PostgreSQL Aggregation Queries

## Introduction

You can fetch aggregations on columns along with nodes using an aggregation query. Aggregations include fields such as
`count`, `sum`, `avg`, `max`, `min`, etc.

[//]: # (The **name of the aggregate field** is initially set in the form `<field-name> + _aggregate`.)

[//]: # (Common aggregation functions are `count`, `sum`, `avg`, `max`, `min`, etc. You)

[//]: # (can see the complete specification of the aggregate field in the [API)

[//]: # (reference]&#40;/api-reference/graphql-api/query.mdx#aggregateobject&#41;. Note that not)

[//]: # (all aggregation functions are available for all data types.)

[//]: # (:::info Note)

[//]: # ()
[//]: # (For more advanced use cases, you can use [views]&#40;/schema/postgres/views.mdx&#41; or)

[//]: # ([custom SQL functions]&#40;/schema/postgres/custom-functions.mdx&#41;.)

[//]: # ()
[//]: # (:::)

## Fetch aggregated data of an object

**Example:** Fetch a list of articles with aggregated data of their rating:

<GraphiQLIDE

  articles_aggregate {
    aggregate {
      count
      sum {
        rating

      avg {
        rating

      max {

      }
    }
    nodes {
      id
      title
      rating
    }
  }
}`}

## Fetch aggregated data on nested objects

The following is an example of a nested object query with aggregations on the **array relationship** between an author
and articles.

**Example:** Fetch author with id "1" and a nested list of articles with aggregated data of their rating:

<GraphiQLIDE
  query={`query {
  authors (where: {id: {_eq: 1}}) {
    id
    name
    articles_aggregate {
      aggregate {
        count
        avg {
          rating
        }
        max {
          rating
        }
      }
      nodes {
        id
        title
        rating
      }
    }
  }
}`}
  response={`{
  "data": {
    "authors": [
      {
        "id": 1,
        "name": "Justin",
        "articles_aggregate": {
          "aggregate": {
            "count": 2,
            "avg": {
              "rating": 2.5
            },
            "max": {
              "rating": 4
            }
          },
          "nodes": [
            {
              "id": 15,
              "title": "vel dapibus at",
              "rating": 4
            },
            {
              "id": 16,
              "title": "sem duis aliquam",
              "rating": 1
            }
          ]
        }
      }
    ]
  }
}`}
/>
