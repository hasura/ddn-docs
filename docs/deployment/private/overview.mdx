# Private Deployment

import { OverviewPlainCard } from "@site/src/components/OverviewPlainCard";

With Private deployment for Hasura DDN, you get to run Hasura and your connectors either on 
dedicated infrastructure hosted by Hasura or on your own infrastructure.
Private Hasura DDN offers enhanced security and isolation by enabling private
connectivity for your databases, APIs and other connectors. Hasura will
be able to talk to your sources over dedicated private network, without
going through the public internet.
