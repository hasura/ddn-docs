# DDN CLI: ddn supergraph build reactivate

Reactivate a hibernated Supergraph build.

## Synopsis

Reactivate a hibernated Supergraph build

```bash
ddn supergraph build reactivate <supergraph-build-version> [flags]
```

## Examples

```bash
# Reactivate a SupergraphBuild for project "pet-lion-2649"
 ddn supergraph build reactivate <supergraph-build-version> --project pet-lion-2649
```

## Options

```sass
    --ci               Disables the use of context
-c, --context string   Name of the context to use. (default <current_context>)
-h, --help             help for reactivate
-p, --project string   DDN Project name
```
