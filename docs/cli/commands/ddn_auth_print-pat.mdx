# DDN CLI: ddn auth print-pat

Prints the PAT to STDOUT.

## Synopsis

Prints the PAT to STDOUT

```bash
ddn auth print-pat [flags]
```

**Alias:** pp

## Examples

```bash
# Print the PAT as a string to STDOUT
 ddn auth print-pat

# Print PAT as a JSON to STDOUT
 ddn auth print-pat --out json
```
