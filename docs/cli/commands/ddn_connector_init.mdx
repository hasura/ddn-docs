# DDN CLI: ddn connector init

Add a new Connector.

## Synopsis

Add a new Connector

```bash
ddn connector init <connector-name> --hub-connector <connector-type> [flags]
```

## Examples

```bash
# Initialize a Postgres Connector "pg_db" inside the directory for Subgraph "app"
ddn connector init pg_db --subgraph app --hub-connector hasura/postgres

# Initialize a Postgres Connector "pg_db" inside the directory ./connector
ddn connector init pg_db --dir ./connector --hub-connector hasura/postgres
```

## Options

```sass
    --dir string             Directory to initialize the Connector
-h, --help                   help for init
    --hub-connector string   Name and version of Connector in Hasura Connector Hub. ref: https://hasura.io/connectors (required)
    --subgraph string        Subgraph to initialize the Connector in
    --supergraph string      Path to Supergraph config file
```
