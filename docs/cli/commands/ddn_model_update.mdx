# DDN CLI: ddn model update

Update a Model in the local metadata.

## Synopsis

Update a Model in the local metadata

```bash
ddn model update <model-name> [flags]
```

## Examples

```bash
# Update the Model "Album" in the Subgraph "app"
 ddn model update Album --subgraph app
```

## Options

```sass
-h, --help                help for update
    --subgraph string     Subgraph that contains the Model
    --supergraph string   Path to Supergraph config file
```
