# Hasura DDN CLI: ddn get connector-build

## Synopsis
Get details of a ConnectorBuild

```bash
$ ddn get connector-build --help
Get details of a ConnectorBuild

Usage:
  ddn get connector-build [id] [flags]

Aliases:
  connector-build, cb, ConnectorBuild

Examples:
 # View details of all ConnectorBuilds in subgraph "app"
   ddn get connector-build --subgraph app

 # Get details of ConnectorBuild with ID "123"
   ddn get connector-build 123

 # Get details of all ConnectorBuilds for ConnectorManifest "mydb"
   ddn get connector-build --connector-manifest mydb

 # Get NDC Schema of ConnectorBuild with ID "123"
   ddn get connector-build 123 --schema


Flags:
      --connector-manifest string   ConnectorManifest to show the information of
  -h, --help                        help for connector-build
      --schema                      Show NDC schema
  -s, --subgraph string             Subgraph to show the ConnectorBuild information of

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
      --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
      --supergraph-manifest string   SupergraphManifest to use
```
