# Hasura3 CLI: hasura3 cloud namespace 

## Synopsis
Namespaces in Hasura Projects.

```bash
$ hasura3 cloud namespace --help
Namespaces in Hasura Projects

Usage: hasura3 cloud namespace [OPTIONS] [COMMAND]

Commands:
  create  Create an environment
  list    List all namespaces of a Hasura Project [aliases: ls]
  remove  Remove a namespace from Hasura Cloud Project [aliases: delete, rm]
  help    Print this message or the help of the given subcommand(s)

Options:
      --log-level <log-level>  The log level to be printed in stdout [env: HASURA_CLI_LOG_LEVEL=]
                               [possible values: error, warn, info, debug, trace, error-all,
                               warn-all, info-all, debug-all, trace-all]
  -h, --help                   Print help
```
