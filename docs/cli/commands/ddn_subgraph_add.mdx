# DDN CLI: ddn subgraph add

Add a Subgraph to the Supergraph.

## Synopsis

Add a Subgraph to the Supergraph

```bash
ddn subgraph add [flags]
```

## Examples

```bash
# Add a Subgraph "./app/subgraph.yaml" to the Supergraph config file "./supergraph.local.yaml"
 ddn subgraph add --subgraph ./app/subgraph.yaml --target-supergraph ./supergraph.local.yaml

# Add a Subgraph "./app/subgraph.yaml" to the Supergraph config files "./supergraph.local.yaml" and "./supergraph.cloud.yaml"
 ddn subgraph add --subgraph ./app/subgraph.yaml --target-supergraph ./supergraph.local.yaml --target-supergraph ./supergraph.cloud.yaml
```

## Options

```sass
-h, --help                            help for add
    --subgraph string                 Path to Subgraph config file (required)
    --target-supergraph stringArray   Supergraph config file to add the Subgraph. Can be repeated to provide multiple Supergraph config files (required)
```
