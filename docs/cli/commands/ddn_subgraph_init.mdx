# DDN CLI: ddn subgraph init

Initialize a new Subgraph in local metadata.

## Synopsis

Initialize a new Subgraph in local metadata

```bash
ddn subgraph init <subgraph-name> [flags]
```

## Examples

```bash
# Initialize a Subgraph "app"
 ddn subgraph init app --supergraph supergraph.yaml
```

## Options

