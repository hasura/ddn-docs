# Hasura DDN CLI: ddn add 

## Synopsis
Add objects to the local metadata.

```bash
$ ddn add --help
Add objects to the local metadata

Usage:
  ddn add [command]

Examples:
 # Add a ConnectorManifest to the local metadata
   ddn add connector-manifest <connector-name> --type <connector-type> --hub-connector <connector-hub-ref> --subgraph <subgraph-name>


Available Commands:
  command            Add a new Command to the local metadata
  connector-manifest Add a new ConnectorManifest to the local metadata
  model              Add a new Model to the local metadata

Flags:
  -h, --help   help for add

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
  -o, --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
  -s, --supergraph-manifest string   SupergraphManifest to use.

Use "ddn add [command] --help" for more information about a command.
```
