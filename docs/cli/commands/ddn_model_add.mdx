# DDN CLI: ddn model add

Add a new Model to the local metadata.

## Synopsis

Add a new Model to the local metadata

```bash
ddn model add --connector-link <connector-link-name> --name <collection-name> [flags]
```

## Examples

```bash
# Add a new Model "Album" from the collection "Album" in the DataConnectorLink "mydb" in Subgraph "app"
 ddn model add --connector-link mydb --name Album --subgraph app
```

## Options

```sass
    --connector-link string   Name of the DataConnectorLink (required)
-h, --help                    help for add
    --name string             Name of the collection to add as a Model (required)
    --subgraph string         Subgraph that contains the DataConnectorLink
    --supergraph string       Path to Supergraph config file
```
