# DDN CLI: ddn doctor

Check if the dependencies of DDN CLI are present.

## Synopsis

Check if the dependencies (Docker and Docker Compose) of DDN CLI are installed, are of the required version and if they are running.

```bash
ddn doctor [flags]
```
