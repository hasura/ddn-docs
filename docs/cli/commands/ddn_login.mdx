# DDN CLI: ddn login

Login to DDN.

## Synopsis

Login to DDN

```bash
ddn login [flags]
```

## Examples

```bash
 # Login with browser
 ddn login

 # Login with Personal Access Token
 ddn login --pat <your-personal-access-token>
```

## Options

```sass
-h, --help         help for login
    --pat string   Personal Access token [env: HASURA_DDN_PAT]
```
