# Hasura DDN CLI: ddn apply

## Synopsis
Apply a resource to its Project on Hasura DDN

```bash
$ ddn apply --help
Apply a resource to its Project on Hasura DDN

Usage:
  ddn apply [command]

Examples:
 # Apply a SupergraphBuild to its Project on Hasura DDN
   ddn apply supergraph-build <build-version>


Available Commands:
  supergraph-build Apply a SupergraphBuild to its Project on Hasura DDN

Flags:
  -h, --help   help for apply

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
      --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
      --supergraph-manifest string   SupergraphManifest to use

Use "ddn apply [command] --help" for more information about a command.
```
