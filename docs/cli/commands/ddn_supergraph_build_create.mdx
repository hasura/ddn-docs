# DDN CLI: ddn supergraph build create

Create a SupergraphBuild on Hasura DDN.

## Synopsis

Create a SupergraphBuild on Hasura DDN

```bash
ddn supergraph build create [flags]
```

## Examples

```bash
# Build the Supergraph for Project "pet-lion-2649"
 ddn supergraph build create --supergraph supergraph.cloud.yaml --project pet-lion-2649

# Build the Supergraph for Project "pet-lion-2649" with a custom subgraph env file and env vars for Subgraph "myapp"
 ddn supergraph build create --supergraph supergraph.cloud.yaml --project pet-lion-2649 --subgraph-env-file myapp:./myapp/.env.myapp.cloud --subgraph-env myapp:foo=bar
```

## Options

```sass
    --description string              (Optional) description of the build
-h, --help                            help for create
-p, --project string                  DDN Project name
    --subgraph-env stringArray        Subgraph environment variable. Should be prefixed with the Subgraph name followed by a colon e.g. "myapp:key=val". Can be repeated to provide multiple Subgraph env vars
    --subgraph-env-file stringArray   Path to Subgraph .env file. Should be prefixed with the Subgraph name followed by a colon e.g. "myapp:./myapp/.env". Can be repeated to provide multiple subgraph env files
    --supergraph string               Path to Supergraph config file
```
