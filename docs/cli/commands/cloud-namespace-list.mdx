# Hasura3 CLI: hasura3 cloud namespace list 

## Synopsis
List all namespaces of a Hasura Project.

```bash
$ hasura3 cloud namespace list --help
List all namespaces of a Hasura Project

Usage: hasura3 cloud namespace list [OPTIONS] <--project <project_config>|--pid <project_id>>

Options:
  -p, --project <project_config>  Path to your Project Config (hasura.yaml) file or a directory
                                  containing that file
      --pid <project_id>          Your Hasura Cloud Project ID
      --log-level <log-level>     The log level to be printed in stdout [env: HASURA_CLI_LOG_LEVEL=]
                                  [possible values: error, warn, info, debug, trace, error-all,
                                  warn-all, info-all, debug-all, trace-all]
  -h, --help                      Print help
```
