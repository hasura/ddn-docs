# DDN CLI: ddn plugins

Manage plugins for the CLI.

## Synopsis

The functionality of the CLI can be extended by using plugins. For a list of all available plugins, run ``ddn plugins list``, or visit this repository: https://github.com/hasura/cli-plugins-index.

If you're interested in contributing, please open a PR against this repo.

**Alias:** plugin

## Available operations

- [ddn plugins install](/cli/commands/ddn_plugins_install) - Install a plugin from the index
- [ddn plugins list](/cli/commands/ddn_plugins_list) - List all available plugins with index, versions and installation status
- [ddn plugins uninstall](/cli/commands/ddn_plugins_uninstall) - Uninstall a plugin
- [ddn plugins upgrade](/cli/commands/ddn_plugins_upgrade) - Upgrade a plugin to a newer version
