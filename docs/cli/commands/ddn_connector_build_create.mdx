# DDN CLI: ddn connector build create

Create a ConnectorBuild on Hasura DDN.

## Synopsis

Create a ConnectorBuild on Hasura DDN

```bash
ddn connector build create --connector <path-to-connector-config-file> [flags]
```

## Examples

```bash
# Create ConnectorBuild from Connector located at ./foo/my_db.connector.yaml
 ddn connector build create --connector ./foo/my_db.connector.yaml --project pet-lion-2649

# Create ConnectorBuild from Connector located at ./foo/my_db.connector.yaml and update schema of DataConnectorLink "mydb"
 ddn connector build create --connector ./foo/my_db.connector.yaml --target-connector-link mydb --target-supergraph ./supergraph.yaml --project pet-lion-2649
```

## Options

```sass
-c, --connector string               Path to Connector config file
-e, --env stringArray                Environment variable, e.g. FOO=bar. Can be repeated to provide multiple env vars
-h, --help                           help for create
-p, --project string                 DDN Project name
    --target-connector-link string   DataConnectorLink to update with the schema from the ConnectorBuild
    --target-supergraph string       Path to Supergraph config file containing target DataConnectorLink
```
