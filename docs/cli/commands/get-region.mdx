# Hasura DDN CLI: ddn get region 

## Synopsis
List resources or get details of a resource from Hasura DDN.

```bash
$ ddn get region --help
List resources or get details of a resource from Hasura DDN

Usage:
  ddn get [command]

Examples:
 # List all your Projects
   ddn get project
		
 # Get details of a specific Project
   ddn get project <project-name>


Available Commands:
  connector-build  List Connector Builds or get details of a specific one
  project          List Projects or get details of a specific one
  subgraph         List Subgraphs or get details of a specific one
  supergraph-build List SupergraphBuilds or get details of a specific one

Flags:
  -h, --help   help for get

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
  -o, --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
  -s, --supergraph-manifest string   SupergraphManifest to use.

Use "ddn get [command] --help" for more information about a command.
```
