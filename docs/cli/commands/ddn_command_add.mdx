# DDN CLI: ddn command add

Add a new Command to the local metadata.

## Synopsis

Add a new Command to the local metadata

```bash
ddn command add --connector-link <connector-link-name> --name <procedure/function-name> [flags]
```

## Examples

```bash
# Add a new Command "Login" from the procedure "Login" in the DataConnectorLink "actions" in "app" Subgraph config
 ddn command add --connector-link actions --name Login --subgraph ./app/subgraph.yaml
```

## Options

```sass
    --connector-link string   Name of the DataConnectorLink (required)
-h, --help                    help for add
    --name string             Name of the procedure or function to add as a Command (required)
    --subgraph string         Path to Subgraph config file
```
