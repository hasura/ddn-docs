# Hasura3 CLI: hasura3 cloud environment create 

## Synopsis
Create an environment.

```bash
$ hasura3 cloud environment create --help
Create an environment

Usage: hasura3 cloud environment create [OPTIONS] --project <project_config> --name <name>

Options:
  -p, --project <project_config>   Path to your Project Config (hasura.yaml) file or a directory
                                   containing that file
      --name <name>                The name of your environment
      --description <description>  The description of your environment
      --log-level <log-level>      The log level to be printed in stdout [env:
                                   HASURA_CLI_LOG_LEVEL=] [possible values: error, warn, info,
                                   debug, trace, error-all, warn-all, info-all, debug-all,
                                   trace-all]
  -h, --help                       Print help
```
