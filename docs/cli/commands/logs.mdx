# Hasura DDN CLI: ddn logs

## Synopsis
Show logs of a connector

```bash
$ ddn logs --help
Show logs of a connector

Usage:
  ddn logs [command]

Examples:
 # Show logs for a ConnectorBuild
   ddn logs connector-build <ConnectorBuild ID>


Available Commands:
  connector-build Get logs of a deployed or built cloud ConnectorBuild

Flags:
  -h, --help   help for logs

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
      --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
      --supergraph-manifest string   SupergraphManifest to use

Use "ddn logs [command] --help" for more information about a command.
```
