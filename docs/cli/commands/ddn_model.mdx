# DDN CLI: ddn model

Perform Model related operations.

## Synopsis

Perform Model related operations

**Alias:** models

## Available operations

- [ddn model add](/cli/commands/ddn_model_add) - Add new Models to the local metadata
- [ddn model list](/cli/commands/ddn_model_list) - Lists details about the collections of DataConnectorLink, and their corresponding Models
- [ddn model remove](/cli/commands/ddn_model_remove) - Removes Models (and related metadata) in the local metadata
- [ddn model update](/cli/commands/ddn_model_update) - Update Models in the local metadata
