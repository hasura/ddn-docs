# DDN CLI: ddn supergraph build local

Build the Supergraph to run the local Engine.

## Synopsis

Build the Supergraph to run the local Engine

```bash
ddn supergraph build local --output-dir <path-to-engine-directory> [flags]
```

## Examples

```bash
# Build the Supergraph using supergraph.yaml file present in current directory and output it to engine directory
 ddn supergraph build local --output-dir <path-to-engine-directory>

# Build the Supergraph using a different supergraph config
 ddn supergraph build local --output-dir <path-to-engine-directory> --supergraph <path-to-supergraph-config-file>
```

## Options

```sass
-e, --env stringArray     Environment variable, e.g. FOO=bar. Can be repeated to provide multiple env vars
-h, --help                help for local
    --output-dir string   Path to the engine directory to output the build artifacts (required)
    --supergraph string   Path to Supergraph config file
```
