# Hasura DDN CLI: ddn update

## Synopsis
Update objects in the local metadata

```bash
$ ddn update --help
Update objects in the local metadata

Usage:
  ddn update [command]

Examples:
 # Update the build context of a ConnectorManifest "mypg"
   ddn update connector-manifest mypg

 # Update DataConnectorLink "mypg" 
   ddn update data-connector-link mypg


Available Commands:
  command             Update a Command in the local metadata
  connector-manifest  Update the build context of a ConnectorManifest.
  data-connector-link Updates the schema of a DataConnectorLink
  model               Update a Model in the local metadata

Flags:
  -h, --help   help for update

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
      --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
      --supergraph-manifest string   SupergraphManifest to use

Use "ddn update [command] --help" for more information about a command.
```
