# Hasura DDN CLI: ddn get cloud-environment-variable 

## Synopsis
See docs here: https://hasura.io/docs/3.0/ci-cd/secrets/.

```bash
$ ddn get cloud-environment-variable --help
See docs here: https://hasura.io/docs/3.0/ci-cd/secrets/

Usage:
  ddn get cloud-environment-variable [flags]

Aliases:
  cloud-environment-variable, CloudEnvironmentVariable, cloudenvvar, cloudenvironmentvariable

Examples:
 # Get a Cloud Environment Variable of a project
   ddn get cloud-environment-variable <key> --project <project-name>
 
 # List all Cloud Environment Variable belonging to a project
   ddn get cloud-environment-variable --project <project-name>

 # Get the value of the Cloud Environment Variable in the default environment (across all subgraphs) of a project
   ddn get cloud-environment-variable <key> --project <project-name> --environment default

 # Get the value of the Cloud Environment Variable in the default subgraph (across all environments) of a project
   ddn get cloud-environment-variable <key> --project <project-name> --subgraph default

 # Get the the value of the Cloud Environment Variable in the default environment in the default subgraph
   ddn get cloud-environment-variable <key> --project <project-name> --subgraph default --environment default

Flags:
  -e, --environment string   Environment of secret
  -h, --help                 help for cloud-environment-variable
  -n, --subgraph string      Subgraph of secret

Global Flags:
      --dir string                   DDN project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. accepts DEBUG, WARN, INFO, ERROR, FATAL (default "INFO")
      --no-prompt                    Set this flag to no prompt for anything, the command fails if any input is missing
  -o, --out string                   Format in which the output should be printed, accepts table, json and yaml (default "table")
  -p, --project string               DDN Project Name
  -s, --supergraph-manifest string   Supergraph manifest to use. (Optional. DefaultSupergraphManifest used  if not provided)
```
