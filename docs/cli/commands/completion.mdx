# Hasura DDN CLI: ddn completion 

## Synopsis
Generate the autocompletion script for ddn for the specified shell..

```bash
$ ddn completion --help
Generate the autocompletion script for ddn for the specified shell.
See each sub-command's help for details on how to use the generated script.

Usage:
  ddn completion [command]

Available Commands:
  bash        Generate the autocompletion script for bash
  fish        Generate the autocompletion script for fish
  powershell  Generate the autocompletion script for powershell
  zsh         Generate the autocompletion script for zsh

Flags:
  -h, --help   help for completion

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
  -o, --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
  -s, --supergraph-manifest string   SupergraphManifest to use.

Use "ddn completion [command] --help" for more information about a command.
```
