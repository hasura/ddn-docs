# Hasura DDN CLI: ddn logs connector-build

## Synopsis
Get logs of a deployed or built cloud ConnectorBuild

```bash
$ ddn logs connector-build --help
Get logs of a deployed or built cloud ConnectorBuild

Usage:
  ddn logs connector-build [ConnectorBuild ID] [flags]

Aliases:
  connector-build, cb, ConnectorBuild

Examples:
 # Get running deploy logs since a time
   ddn logs connector-build <ConnectorBuild ID> --since 2024-03-26T10:19:51.048844Z
 # Get running deploy logs and keep following since a specified time duration
   ddn logs connector-build <ConnectorBuild ID> --follow --since 2m
 # Get build logs of a deployment
   ddn logs connector-build <ConnectorBuild ID> --build


Flags:
      --build          Specifies whether to show the build logs. Default is false.
      --follow         Specifies whether to continuously follow the deployment logs. Default is false.
  -h, --help           help for connector-build
      --since string   Specifies the starting time for log retrieval. Can be in ISO format (e.g., 2024-03-26T11:05:15Z) or a duration (e.g., 5m for 5 minutes ago). Maximum allowed duration is 10m from now for deploy logs. Default is 10m for deploy logs.

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
      --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
      --supergraph-manifest string   SupergraphManifest to use
```
