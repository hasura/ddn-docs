# DDN CLI: ddn command update

Update a Command in the local metadata.

## Synopsis

Update a Command in the local metadata

```bash
ddn command update <command-name> [flags]
```

## Examples

```bash
# Update the Command "Login" in Subgraph "app"
ddn command update Login --subgraph app
```

## Options

```sass
-h, --help                help for update
    --subgraph string     Subgraph that contains the Command
    --supergraph string   Path to Supergraph config file
```
