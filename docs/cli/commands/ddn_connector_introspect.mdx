# DDN CLI: ddn connector introspect

Introspect the Connector data source to update the Connector configuration.

## Synopsis

Introspect the Connector data source to update the Connector configuration

```bash
ddn connector introspect --connector <path-to-connector-config-file> [flags]
```

## Examples

```bash
# Introspect Connector located at ./foo/my_db.connector.yaml
 ddn connector introspect --connector ./foo/my_db.connector.yaml
```

## Options

```sass
-c, --connector string   Path to Connector config file
-e, --env stringArray    Environment variable, e.g. FOO=bar. Can be repeated to provide multiple env vars
-h, --help               help for introspect
```
