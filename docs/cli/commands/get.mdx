# Hasura DDN CLI: ddn get 

## Synopsis
List resources or get details of a specific resource from Hasura DDN.

```bash
$ ddn get --help
List resources or get details of a specific resource from Hasura DDN

Usage:
  ddn get [command]

Examples:
 # List all your Projects
   ddn get project
		
 # Get details of a Project "pet-lion-2649"
   ddn get project pet-lion-2649


Available Commands:
  connector-build  Get details of a ConnectorBuild
  project          List Projects or get details of a specific one
  subgraph         List Subgraph or get details of a specific one
  supergraph-build List SupergraphBuilds or get details of a specific one

Flags:
  -h, --help   help for get

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
      --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
      --supergraph-manifest string   SupergraphManifest to use

Use "ddn get [command] --help" for more information about a command.
```
