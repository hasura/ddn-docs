# Hasura DDN CLI: ddn add model 

## Synopsis
Add a new Model to the local metadata.

```bash
$ ddn add model --help
Add a new Model to the local metadata

Usage:
  ddn add model [flags]

Examples:
 # Add a new Model 'Album' from the collection 'Album' in the DataConnectorLink 'mypg'
   ddn add model --data-connector-link mypg --name Album
	 

Flags:
      --data-connector-link string   name of the data connector link
  -h, --help                         help for model
      --name string                  name of the collection to add as a model to the metadata
      --subgraph string              name of the Subgraph to add the connector to

Global Flags:
      --dir string                   DDN Project directory in which hasura.yaml is present (default ".")
      --log-level string             Log level. Can be DEBUG, WARN, INFO, ERROR, or FATAL. (default "INFO")
      --no-prompt                    Do not prompt for required but missing flags
  -o, --out string                   Output format. Can be table, json or yaml. (default "table")
  -p, --project string               DDN Project name
  -s, --supergraph-manifest string   SupergraphManifest to use.
```
