# Hasura3 CLI: hasura3 namespace 

## Synopsis
Manage Hasura project namespace [aliases: ns, namespaces].

```bash
$ hasura3 namespace --help
Manage Hasura project namespace [aliases: ns, namespaces]

Usage:
  hasura3 namespace [command]

Aliases:
  namespace, ns, namespaces

Available Commands:
  create      Create a namespace
  delete      Remove a namespace from Hasura Cloud Project [aliases: remove, rm]
  describe    View namespace details [aliases: details, info]
  list        List all namespaces in a Hasura project [alias: ls]

Flags:
  -h, --help             help for namespace
  -p, --project string   Hasura DDN Project Name

Global Flags:
      --dir string         Hasura project directory in which hasura.yaml is present (default ".")
      --log-level string   Log level. accepts DEBUG, WARN, INFO, ERROR, FATAL (default "INFO")
      --no-prompt          set this flag to no prompt for anything, the command fails if any input is missing
  -o, --out string         format in which the output should be printed, accepts table, json and yaml (default "table")

Use "hasura3 namespace [command] --help" for more information about a command.
```
