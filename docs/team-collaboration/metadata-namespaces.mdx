# Metadata Namespaces

A metadata with namespaces looks something like the following:

```json
[
  {
    "namespace": "foo",
    "metadata": [
      {<metadata obj>},
      {<metadata obj>}
    ]
  },
  {<metadata with namespace},
  ...
]
```
