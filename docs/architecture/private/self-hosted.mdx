# Self-Hosted

## Introduction

For customers with strict security and compliance requirements, Private DDN Self Hosted allows you
to host both Control Plane and Data Plane on your own infrastructure. 

This is a premium offering where Hasura team will be helping you with setting up the entire DDN
Platform organization wide.

If you would like access to Private DDN Self Hosted, please [contact sales](https://hasura.io/contact-us).

## Architecture

In the Self Hosted mode, Control Plane and Data Plane is running on customer's cloud account, managed
by customer's own operations team, with the help of Hasura team.
Uptime and reliability of the data plane is the responsibility of the customer's infrastrucure team.

A Kubernetes cluster is required for installing DDN.
