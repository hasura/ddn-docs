# DDN CLI: ddn auth login

Login to DDN.

## Synopsis

Login to DDN

```bash
ddn auth login [flags]
```

## Examples

```bash
# Login with browser
 ddn auth login

# Login with Personal Access Token
 ddn auth login --pat <your-personal-access-token>
```

## Options

```sass
-h, --help         help for login
    --pat string   Personal Access token [env: HASURA_DDN_PAT]
```
